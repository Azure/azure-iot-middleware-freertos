Detailed Description

◆ AzureIoTHubClient_GetDeviceTwin()

AzureIoTResult_t AzureIoTHubClient_GetDeviceTwin ( AzureIoTHubClient_t *  pxAzureIoTHubClient)

Request to get the device twin document.

Note
AzureIoTHubClient_SubscribeDeviceTwin() must be called before calling this function.

The response to the request will be returned via the AzureIoTHubClientTwinCallback_t which was passed in the AzureIoTHubClient_SubscribeDeviceTwin() call. The type of message will be eAzureIoTHubTwinGetMessage and the payload (on success) will be the twin document.

Parameters
[in]pxAzureIoTHubClientThe AzureIoTHubClient_t * to use for this call.
Returns
