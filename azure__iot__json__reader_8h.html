◆ AzureIoTJSONReader_SkipChildren()

AzureIoTHubClientResult_t AzureIoTJSONReader_SkipChildren ( AzureIoTJSONReader_t *  pxReader)

Reads and skips over any nested JSON elements.

Parameters
[in]pxReaderA pointer to an #AzureIoTJSONReader_t instance containing the JSON to read.
Returns
An #AzureIoTHubClientResult_t value indicating the result of the operation.
Return values
eAzureIoTHubClientSuccessThe children of the current JSON token are skipped successfully.
Remarks
If the current token kind is a property name, the reader first moves to the property value. Then, if the token kind is start of an object or array, the reader moves to the matching end object or array. For all other token kinds, the reader doesn't move and returns eAzureIoTHubClientSuccess.

Returns
